Server side application: calculator-server.py
Client side testing application: calculator-testing-client.py

Testing data has been hard coded in the client side testing application.
data example: [2, 4, '3+12', 6, '1+12/3']

In command line, first type ’python calculator-server.py’ to run server application. Then type ‘python calculator-testing-client.py’ to run client testing application as many times as you want.
